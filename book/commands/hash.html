Apply hash function.

# Usage

> hash ...args <subcommand> {flags}

# Subcommands

  • hash base64 - base64 encode or decode a value
  • hash md5 - md5 encode a value

# Parameters

  • ...args: optionally convert by column paths

# Flags

  • -h, --help: Display this help message
