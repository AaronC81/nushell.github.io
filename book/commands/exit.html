    • Command Reference

Runs a script file in the current context.

# Signature

> exit (exit_code) --now

# Parameters

  • exit_code: Exit code to return immediately with
  • --now: Exit out of the shell immediately

# Examples

Exit the current shell

> exit

Exit all shells (exiting Nu)

> exit --now
