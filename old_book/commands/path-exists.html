Check whether a path exists

# Usage

> path exists ...args {flags}

# Parameters

  • ...args: Optionally operate by column path

# Flags

  • -h, --help: Display this help message

# Examples

Check if a file exists

> echo '/home/joe/todo.txt' | path exists
