Finds the sum of a list of numbers or tables

# Usage

> math sum {flags}

# Flags

  • -h, --help: Display this help message

# Examples

Sum a list of numbers

> echo [1 2 3] | math sum

Get the disk usage for the current directory

> ls --all --du | get size | math sum
